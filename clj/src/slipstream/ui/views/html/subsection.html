Subsection Template

Currently, subsections are implemented as tabs in form of stacked pills. However, this template offers an abstraction as a subsection, so that the underlying layout can be changed without modifiyng the enlive code.

Other options include:

Class Description
.active Add .active to the ul > li and div.tab-pane to open it by default.

From: http://getbootstrap.com/javascript/#tabs and http://getbootstrap.com/components/#nav
