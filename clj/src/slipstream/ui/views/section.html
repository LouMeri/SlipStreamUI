Section Template

Currently, sections are implemented as accordions. However, this template offers an abstraction as a section, so that the underlying layout can be changed without modifiyng the enlive code.

Other options include:

Class Description
.in Add .in to the div.panel-collapse to open it by default. Remember to update the .glyphicon-chevron-up accordingly.
.panel-info Add .panel-info to the div.panel of the section.
.panel-danger Add .panel-danger to the div.panel of the section.
.panel-success Add .panel-success to the div.panel of the section.
